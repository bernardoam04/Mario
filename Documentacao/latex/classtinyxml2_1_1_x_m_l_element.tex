Convenience function for easy access to the text inside an element. Although easy and concise, \doxylink{classtinyxml2_1_1_x_m_l_element_a1f9c2cd61b72af5ae708d37b7ad283ce}{Set\+Text()} is limited compared to creating an \doxylink{classtinyxml2_1_1_x_m_l_text}{XMLText} child and mutating it directly.

If the first child of \textquotesingle{}this\textquotesingle{} is a \doxylink{classtinyxml2_1_1_x_m_l_text}{XMLText}, \doxylink{classtinyxml2_1_1_x_m_l_element_a1f9c2cd61b72af5ae708d37b7ad283ce}{Set\+Text()} sets its value to the given string, otherwise it will create a first child that is an \doxylink{classtinyxml2_1_1_x_m_l_text}{XMLText}.

This is a convenient method for setting the text of simple contained text\+: \begin{DoxyVerb}<foo>This is text</foo>
    fooElement->SetText( "Hullaballoo!" );
<foo>Hullaballoo!</foo>
\end{DoxyVerb}

then it will not change "{}\+This is text"{}, but rather prefix it with a text element\+: \begin{DoxyVerb}    <foo>Hullaballoo!<b>This is text</b></foo>
\end{DoxyVerb}
